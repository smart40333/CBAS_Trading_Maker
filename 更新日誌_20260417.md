# 更新日誌 2026-04-17

本次更新主要針對 **VIP名單 / 特殊報價** 兩個分頁，以及 **暫存檔** 與 **CB自營系統比對** 的體驗優化。

---

## 🆕 新功能

### 1. VIP名單：新增「80元手續費」欄位
VIP名單表格多了一欄 **「80元手續費」**（Y/N）。
- 若填 **Y**：該客戶的手續費固定以 **80 元** 計算
- 若填 **N**：維持原本邏輯（照「不限張數低手續費」或基礎規則）

**畫面提示**：填了 Y 的格子會自動上**淺藍色**，方便一眼看出誰適用。

**手續費優先順序**（由高至低）：
1. 特殊報價（最優先）
2. **80元手續費 = Y → 80**（本次新增）
3. 不限張數低手續費 = Y → 100
4. 特殊客戶ID → 60
5. 基礎規則 → 100 / 110 / 150

---

### 2. 特殊報價：新增「張數」「備註」欄位與自動扣減
特殊報價表格多了兩欄：**張數** 和 **備註**。

#### 「張數」用法
- **留空**：代表**無張數限制**（維持原本全量套用 VIP 報價）
- **填入正整數 N**：代表該 VIP 報價**只適用 N 張**
- **填 0**：等同於作廢，該筆不套用 VIP 報價

#### 拆單邏輯（舉例）
假設 客戶A + 2330 特殊報價張數為 **50 張**，今天客戶A 交易 2330 共 **100 張**：
- **前 50 張** → 套 VIP 報價（特殊利率/手續費）
- **剩 50 張** → 照原本邏輯（VIP名單/特殊ID/基礎規則）
- 產生的上傳檔會變成**兩筆契約**（新作契約編號不同）

如果客戶A 只交易 **30 張**（比 VIP 張數少）：
- **全部 30 張** 都套 VIP 報價（只產一筆契約）
- 剩下 **20 張** 額度會在扣減後保留在 VIP_Quote 中

#### 自動扣減與備註（按下「產生上傳檔」後才生效）
當你按下「**產生上傳檔**」且產檔成功後，系統會自動：
1. 依照這次用掉的張數，從 VIP_Quote 表格對應那筆扣掉
2. 在「備註」欄**追加**一行：`於YYYYMMDD扣除了{X}張`
3. 若張數扣到 **0**，該筆會**自動被刪除**
4. 表格與檔案（`VIP_Quote.csv`）都會即時更新

> ⚠ 注意：只有「產生上傳檔」成功才會扣減。只是開啟、預覽或計算都不會動到張數。

### 3. CB自營系統不符時顯示「是哪幾筆不一樣」
以前按下「張數/總額確認」，若與 CB自營系統金額不符，只會跳一個總數警告。

**現在改為**：
- 會依 **(客戶ID, CB代號) vs (客戶代碼, 商品代碼)** 比對，列出**每一筆**不符的明細，例如：

  ```
  H123456789 / 2330: 中台=50張,5,000,000元  CB自營=45張,4,500,000元  (差 5張, 500,000元)
  ```
  - **中台**＝本中台產檔系統（目前表格）算出來的數字
  - **CB自營**＝CB自營軟體資料庫（`RPT_Monitor_Fill`）的數字
  - **差**＝中台 − CB自營（正數=中台比較多，負數=CB自營比較多）

- 該不符的列會被**紅色標記**（盤面交易列才標）
- 買進 / 提解賣出 都有這個新功能

#### 特殊情況：CB自營有、但中台沒有
如果 CB自營軟體裡有某筆交易，本中台卻沒有（漏算、漏讀），彈窗會顯示：
```
H123456789 / 2330: 中台=0張,0元  CB自營=10張,1,000,000元  (差 -10張, -1,000,000元)
```
- **差為負數** 就代表「CB自營比較多」
- 這種情況因為表格裡根本沒這列 → **紅色標記不會出現**，只能從彈窗看到明細，務必留意

> 賣出側如果某列同時「重複原單契約編號」+「CB自營不符」→ 紅色會蓋過橙色（紅色優先，方便找到需要處理的項目）

---

### 4. 讀取暫存檔會自動上色
以前「讀取暫存」只會把資料填回表格，需要自己再手動按「報價確認」「張數確認」才會上色。

**現在改為**：
- 讀取暫存後，系統會**自動跑檢查**並把顏色套好，不用再按按鈕
- 讀取過程**不會彈任何確認視窗**（安靜執行）
- 顏色結果跟手動按的完全一致：
  - 買進「權利金百元價」：符合=綠、不符=淺紅
  - 賣出「原單契約編號」重複=橙
  - CB自營不符的列=紅

## 🗂 資料檔格式升級（向後相容）

以下兩個檔會自動新增欄位，**舊檔直接開不會壞**：

- **VIP_List.csv**：多一欄 `80元手續費`
- **VIP_Quote.csv**：多兩欄 `張數`、`備註`

首次從 UI 按「儲存」後，CSV 就會補上新欄位。

---

## 💡 建議操作流程

1. 打開 AP → VIP名單 tab → 確認有新的「80元手續費」欄位 → 需要的客戶填 Y
2. 特殊報價 tab → 確認有「張數」「備註」欄位 → 設定每筆的張數上限
3. 按「儲存」讓檔案更新新欄位
4. 正常作業：讀取當日資料 → 確認表格 → 產生上傳檔
5. 產檔成功後 → 切到「特殊報價」tab 可以看到張數被扣、備註被追加

有任何問題或發現異常，請告知。
